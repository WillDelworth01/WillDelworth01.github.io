Web Application Programming and Hacking

WAPH is a hands on course on how the web works and how it breaks. It moves from the foundations of HTTP and front end development through full stack secure development in PHP and MySQL, with a recurring focus on common web attacks and the defenses against them. Everything below is work I built for the course.

01Labs

Lab 0: Environment Setup

Built a LAMP stack Ubuntu VM, Apache server, and a Git and SSH workflow.

Lab 1: Foundations of the Web

Inspected HTTP with Wireshark and telnet, wrote CGI in C and PHP, and compared GET versus POST.

Lab 2: Front End Web Development

HTML forms and JavaScript, then Ajax, CSS, jQuery, and Web API integration with JSON.

Lab 3 and Lab 4: Secure Development

Secure web development in PHP and MySQL, and secure session authentication.

02Hackathons (Attacks and Defenses)

XSS: Cross Site Scripting SQL Injection Session Hijacking CSRF

Each hackathon has two halves: carrying out the attack against a vulnerable app, then implementing the defense (input validation, output encoding, prepared statements, and so on).
